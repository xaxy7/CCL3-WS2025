Project description

Have you ever felt ashamed that you cannot recognize a flag? You wish that you knew more about various nations around the world? Our app is exactly what you need. We allow users to quickly learn every country flag by playing a dynamic quiz game with different modes.

Team members:

Ksawery Kochanowicz & Denis Dubovicky

App concept:

Use case

The app can be used simply for learning different facts about countries all around the world. However, it can have other benefits, such as creating good habits by practicing memorisation every day. It can also serve as a fun distraction to use while bored, and one that actually teaches you some potentially useful facts, instead of brain rotting

Target user

  • Age: Teen/Young Adults, but also Seniors
  • Gender: any
  • Occupation: School and Collage Students,
  • Interests: geography, memory games, politics, daily games

User Testing

Initial Hypothesis

Users will perceive the country shape and flag guessing mobile application as usable, as indicated by System Usability Scale and Single Ease question

Variables

Dependent:

  • Perceived usability measured via System Usability Scale (SUS)
  • Perceived task ease measured via Single Ease Question (SEQ)
  • Qualitative user feedback and observed usability issues

Independent: Version of the application

  • Version A: Initial prototype
  • Version B: Improved prototype

Methods

This study employed an iterative usability evaluation conducted in two tests.

In the first round, an initial version (Version A) of the application was evaluated through task-based usability testing. Based on the findings from this formative evaluation, design improvements were implemented.

In the second round, the improved version (Version B) was evaluated using the same usability testing procedure with a new group of participants.

Usability Test Plan

  1. Informed Consents
  2. Demographic questioner
  3. Single Ease Score
    1. Play Flag Guessing -> Hard Mode - until you complete it or run out of lives
    2. View the summary of the completed round
    3. Select a Country Guessing mode of you choice and play until you bookmark 5 country shapes, then return to the main screen
    4. Remove one of the bookmarks on the country shape and then plan a notebook round
    5. View the History of your games from your test, and remove one of them
  4. SUS questioner
  5. Free, open feedback

Usability Testing Results – GeoQuest

We ran our test twice. Our formative test was conducted with initial version of the application with 5 participants. Feedback and results obtained were used to improve design, functionality and experience of GeoQuest. Improved version was tested in the summative test. The newly recruited 5 participants pefromed the same tasks as the first testing group. Results from both experiments were compared and analyzed. Conclusion We can observe slight increase of 4% in average SEQ between test A (dSEQ = 6,44) and test B (dSEQ = 6,72). Consitent with 3% increase in average SUS score and lower SUS score standart deviation. Summative test – additional information Four out five participants fall into the age group 18-24 and once participant in age group 35-50. Participants were familiar with the flags slightly above average. The average familiarity score = 3.17/5 All of the participants answered correctly when asked to identify three flags. 4 out of 5 participants were familiar with geography apps like Geoguesser or Geotastic. Most of the users in our participants used android as their primary smartphone operating system All tasks were perceived as very easy or easy. Calculated SUS score show that our paticipants rate our app with grade A and A+
